# Módulo 1 - Arquitetura, Ferramentas Administrativas e Configuração do Ambiente

Este pacote contém o material do Módulo 1 dividido em duas partes:

- `01-teoria-modulo1.md` - conteúdo teórico explicado do módulo;
- `02-pratica-modulo1.md` - ambiente, comandos, visões de sistema e roteiro prático.

## Objetivo

Fornecer a base conceitual e operacional para o início do curso de Oracle Administração, preparando professor e alunos para entender a arquitetura do banco, conectar ao ambiente e explorar as principais visões administrativas.
