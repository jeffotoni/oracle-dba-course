# Grupo 2 - Cache Simples Key-Value com Oracle

## Tema

Construir uma ferramenta simples de cache key-value usando Oracle.

## Objetivo

O grupo deve montar uma solução pequena que siga esta linha:

```txt
Cliente -> API simples -> Oracle -> chave / valor / expiração opcional
```

## Ideia do projeto

A proposta é criar um cache simples para armazenar pares de chave e valor no Oracle.

O foco é mostrar:

- modelagem mínima;
- operações simples de gravação, leitura e remoção;
- uso do Oracle em um cenário leve de aplicação.

## O que o trabalho deve demonstrar

- como modelar um cache relacional simples;
- como gravar e consultar por chave;
- como remover registros;
- opcionalmente como tratar expiração.

## Escopo mínimo

### 1. Banco Oracle

Criar uma tabela simples de cache.

Exemplo:

```sql
CREATE TABLE cache_store (
  id          NUMBER GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  cache_key   VARCHAR2(150) NOT NULL UNIQUE,
  cache_value CLOB NOT NULL,
  expires_at  TIMESTAMP,
  created_at  TIMESTAMP DEFAULT SYSTIMESTAMP NOT NULL
);
```

> Observação: `cache_value` pode continuar como `CLOB`, que é a opção mais simples. Se o grupo quiser explorar um formato mais flexível, pode estudar também o uso de `JSON`.

## 2. Operações mínimas

O trabalho deve ter pelo menos:

- criar um item;
- buscar um item por chave;
- remover um item;
- listar itens existentes.

Opcionalmente:

- verificar expiração;
- atualizar valor existente;
- limpar itens expirados.

## 3. API sugerida

Exemplo de endpoints:

- `POST /cache`
- `GET /cache/{key}`
- `DELETE /cache/{key}`
- `GET /cache`

## Exemplos SQL

### Inserir

```sql
INSERT INTO cache_store (cache_key, cache_value, expires_at)
VALUES (
  'user:1001',
  '{"name":"Jefferson","role":"admin"}',
  SYSTIMESTAMP + INTERVAL '10' MINUTE
);
```

### Buscar por chave

```sql
SELECT cache_key,
       cache_value,
       expires_at,
       created_at
FROM cache_store
WHERE cache_key = 'user:1001';
```

### Remover por chave

```sql
DELETE FROM cache_store
WHERE cache_key = 'user:1001';
```

### Listar todos

```sql
SELECT cache_key,
       expires_at,
       created_at
FROM cache_store
ORDER BY created_at DESC;
```

### Buscar só itens válidos

```sql
SELECT cache_key,
       cache_value,
       expires_at
FROM cache_store
WHERE expires_at IS NULL
   OR expires_at > SYSTIMESTAMP
ORDER BY created_at DESC;
```
